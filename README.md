# Site do Campeonato da Igreja

Este é o esboço inicial do site do campeonato de futebol.

## Como usar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos:
   - index.html
   - style.css
   - script.js
   - README.md
3. Vá em **Settings > Pages**
4. Em *Build and Deployment*, escolha:
   - Source: **Deploy from branch**
   - Branch: **main** e pasta **root**
5. Clique em Save.
6. O site ficará disponível em alguns instantes.
